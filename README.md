# 🏥 CareNest

Smart Child Healthcare & Vaccination Management System  
Built with MERN Stack + Docker + MongoDB Atlas

---

## 🚀 Live Features

- 🔐 Role-based Authentication (Parent / Doctor)
- 👶 Child Registration
- 💉 Auto Vaccine Schedule Generation
- 📊 Growth Tracking with BMI Calculation
- 📈 Dashboard Analytics
- 🧑‍⚕️ Doctor Panel (View All Children)
- 📄 PDF Medical Report Download
- 🐳 Fully Dockerized Setup
- ☁️ MongoDB Atlas Cloud Database

---

## 🛠️ Tech Stack

**Frontend**

## 🐳 Run With Docker

```bash
docker compose up --build

App runs on:

Frontend: http://localhost:3000

Backend: http://localhost:5000

🔐Environment Variables

Create a .env file in root:

JWT_SECRET=your_secret_key
